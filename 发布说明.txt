DeepSeek 聊天总结工具 - 发布说明
=======================================

版本：v2.0
发布日期：2025年1月

主要变更：
----------

1. 架构优化
   - 移除了内置的 chatlog 服务管理功能
   - 改为连接外部 chatlog 服务
   - 简化了应用程序结构

2. 配置管理
   - 重新设计配置页面界面
   - DeepSeek API 配置与 Chatlog 服务配置分离
   - 支持动态配置 chatlog 服务地址
   - 配置保存到 config.json 文件

3. 界面改进
   - 修复了配置页面显示空白的问题
   - 优化了布局管理
   - 改进了用户体验

4. 打包优化
   - 移除了 chatlog.exe 的打包
   - 减小了应用程序体积
   - 支持独立部署

使用要求：
----------

1. 系统要求：
   - Windows 10/11
   - 64位系统

2. 服务依赖：
   - 需要单独部署和运行 chatlog 服务
   - 需要 DeepSeek API 密钥

3. 网络要求：
   - 能够访问 DeepSeek API (https://api.deepseek.com)
   - 能够访问本地或远程的 chatlog 服务

安装说明：
----------

1. 下载 DeepSeek聊天总结工具.exe
2. 将文件放置到任意目录
3. 双击运行即可

首次使用：
----------

1. 启动应用程序
2. 切换到"配置"标签页
3. 输入 DeepSeek API 密钥
4. 配置 chatlog 服务地址（默认：http://127.0.0.1:5030/api/v1）
5. 点击"保存配置"
6. 切换到"聊天记录总结"标签页开始使用

注意事项：
----------

- 确保 chatlog 服务已启动并可访问
- API 调用会消耗 DeepSeek 额度
- 聊天记录会发送到 DeepSeek API 进行处理
- 建议在稳定的网络环境下使用

技术支持：
----------

如遇到问题，请检查：
1. chatlog 服务是否正常运行
2. DeepSeek API 密钥是否有效
3. 网络连接是否正常
4. 配置信息是否正确
